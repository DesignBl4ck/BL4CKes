Aplicaciones Web

¿Sabes realmente como funciona internet? ¿Qué es una aplicación web? ¿O como hacer para construir una? Has llegado al curso correcto para contestar estas preguntas y muchas más. Aquí veremos desde los conceptos más básicos acerca de la web hasta iniciarnos en el mundo del desarrollo de aplicaciones web (que dicho sea de paso, es un programa que corre en una PC con un servidor web mientras sus usuarios interactúan con este desde un navegador web ;) Aprenderemos sobre los lenguajes necesarios para empezar, como HTML, CSS y Javascript. Este curso introductorio te abrirá las puertas a convertirte de sólo consumidor a creador de la web. O si tu caso es que ya sabes estos conceptos, es una excelente oportunidad para refrescarlos y compartir tus conocimientos con los iniciantes y así ¡todos ganamos!
